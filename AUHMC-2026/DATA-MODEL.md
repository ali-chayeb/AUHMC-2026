هذا الملف يوثق جميع البيانات والثوابت المستخدمة في الموقع. جميع البيانات مخزنة في ملف `app.js` على شكل مصفوفات (Arrays) وكائنات (Objects). لا توجد قاعدة بيانات خارجية أو اتصال بخادم.

## 🧬 المسارات العلمية (TRACKS)

مصفوفة من الكائنات، كل كائن يمثل مساراً علمياً.

```javascript
const TRACKS = [
  {
    id: 'pediatrics',        // معرف فريد (للإستخدام الداخلي)
    icon: 'fa-baby',         // أيقونة Font Awesome
    title: 'طب الأطفال وحديثي الولادة',  // الاسم العربي
    desc: 'المقاربات الحديثة...'         // الوصف
  },
  // ...
];
```

### الحقول

| الحقل | النوع | الوصف | مثال |
|---|---|---|---|
| `id` | `string` | معرف فريد للمسار (بالإنجليزية) | `'pediatrics'` |
| `icon` | `string` | اسم أيقونة Font Awesome (بدون prefix `fas`) | `'fa-baby'` |
| `title` | `string` | اسم المسار بالعربية | `'طب الأطفال وحديثي الولادة'` |
| `desc` | `string` | وصف المسار بالعربية | `'المقاربات الحديثة للإنتانات...'` |

### المسارات الثمانية

| # | المعرف (id) | الأيقونة | الاسم |
|---|---|---|---|
| 1 | `pediatrics` | `fa-baby` | طب الأطفال وحديثي الولادة |
| 2 | `surgery` | `fa-scalpel` | الجراحة العامة والتخصصية |
| 3 | `internal` | `fa-heartbeat` | الأمراض الباطنية |
| 4 | `diagnostic` | `fa-microscope` | الطب التشخيصي المتقدم |
| 5 | `subspecialties` | `fa-eye` | الاختصاصات الدقيقة |
| 6 | `education` | `fa-chalkboard-teacher` | التعليم الطبي المستمر |
| 7 | `ai` | `fa-robot` | الذكاء الاصطناعي الطبي |
| 8 | `quality` | `fa-shield-alt` | الجودة وسلامة المرضى |

## 📋 البرنامج العلمي (SCHEDULE)

كائن (Object) المفاتيح فيه هي أرقام الأيام (1-5)، وكل مفتاح يقابله مصفوفة من الفعاليات.

```javascript
const SCHEDULE = {
  1: [  // اليوم الأول
    {
      time: '12:00 – 13:30',     // الوقت
      title: 'الافتتاح الرسمي...', // عنوان الفعالية
      speaker: 'رئيس المؤتمر',     // المتحدث (قد يكون فارغاً)
      track: 'عام'                // المسار التابعة له
    },
    // ...
  ],
  2: [ /* ... */ ],
  // ...
};
```

### الحقول

| الحقل | النوع | الوصف | مثال |
|---|---|---|---|
| `time` | `string` | الفترة الزمنية | `'12:00 – 13:30'` |
| `title` | `string` | عنوان الفعالية | `'الافتتاح الرسمي...'` |
| `speaker` | `string` | اسم المتحدث (فارغ إذا لا يوجد) | `'رئيس المؤتمر'` أو `''` |
| `track` | `string` | اسم المسار (يطابق أحد عناوين TRACKS أو 'عام') | `'عام'` |

### ملاحظات
- المسار `'عام'` هو مسار خاص للفعاليات العامة (الافتتاح، الختام)
- قيمة `track` يجب أن تطابق أحد قيم `title` في `TRACKS` أو تكون `'عام'`

## 👥 اللجان (COMMITTEES)

مصفوفة من الكائنات، كل كائن يمثل لجنة من اللجان المنظمة.

```javascript
const COMMITTEES = [
  {
    icon: 'fa-crown',           // أيقونة Font Awesome
    title: 'اللجنة العليا',      // اسم اللجنة
    desc: 'الإشراف العام...'     // وصف مهام اللجنة
  },
  // ...
];
```

### الحقول

| الحقل | النوع | الوصف | مثال |
|---|---|---|---|
| `icon` | `string` | اسم أيقونة Font Awesome | `'fa-crown'` |
| `title` | `string` | اسم اللجنة بالعربية | `'اللجنة العليا'` |
| `desc` | `string` | وصف مهام اللجنة | `'الإشراف العام على المؤتمر...'` |

### اللجان الست

| # | الأيقونة | الاسم | الوصف |
|---|---|---|---|
| 1 | `fa-crown` | اللجنة العليا | الإشراف العام والتوجيه الاستراتيجي |
| 2 | `fa-flask` | اللجنة العلمية | تقييم الأبحاث والبوسترات |
| 3 | `fa-tasks` | اللجنة التنظيمية | التنسيق اللوجستي وإدارة الفعاليات |
| 4 | `fa-truck` | اللجنة اللوجستية | تجهيز القاعات والمعدات |
| 5 | `fa-bullhorn` | اللجنة الإعلامية | التغطية الإعلامية والتسويق |
| 6 | `fa-hand-holding-heart` | لجنة الرعاية | التواصل مع الرعاة والشركاء |

## 🛠️ ورشات العمل (WORKSHOPS)

مصفوفة من الكائنات، كل كائن يمثل ورشة عمل.

```javascript
const WORKSHOPS = [
  {
    name: 'كتابة وقراءة الأبحاث الطبية (EBM)',  // اسم الورشة
    capacity: 30                                  // الطاقة الاستيعابية
  },
  // ...
];
```

### الحقول

| الحقل | النوع | الوصف | مثال |
|---|---|---|---|
| `name` | `string` | اسم الورشة بالعربية | `'كتابة وقراءة الأبحاث الطبية (EBM)'` |
| `capacity` | `number` | الطاقة الاستيعابية (عدد المقاعد) | `30` |

### الورشات

| # | الاسم | الطاقة |
|---|---|---|
| 1 | كتابة وقراءة الأبحاث الطبية (EBM) | 30 |
| 2 | تخطيط صدى القلب السريري (Echocardiography) | 20 |
| 3 | الإنعاش القلبي الرئوي المتقدم (ACLS) | 25 |

## 🏢 الرعاة (SPONSORS)

مصفوفة من الكائنات، كل كائن يمثل راعياً.

```javascript
const SPONSORS = [
  {
    name: 'الراعي الرسمي',     // اسم الراعي
    tier: 'الراعي الماسي',     // التصنيف
    desc: 'شركة رائدة...'      // الوصف
  },
  // ...
];
```

### الحقول

| الحقل | النوع | الوصف | مثال |
|---|---|---|---|
| `name` | `string` | اسم الراعي | `'الراعي الرسمي'` |
| `tier` | `string` | تصنيف الراعي | `'الراعي الماسي'` |
| `desc` | `string` | وصف الراعي | `'شركة رائدة في المجال الدوائي'` |

### الرعاة

| # | الاسم | التصنيف |
|---|---|---|
| 1 | الراعي الرسمي | الراعي الماسي |
| 2 | الراعي الثاني | الراعي الذهبي |
| 3 | الراعي الثالث | الراعي الذهبي |
| 4 | الراعي الرابع | الراعي الفضي |

## 🌙 الوضع الليلي (Dark Mode)

عند تفعيل الوضع الليلي (بإضافة كلاس `dark-mode` على `<body>`)، يتم إعادة تعريف المتغيرات التالية:

```css
body.dark-mode {
  --primary: #1a3a7a;
  --primary-light: #2a4a8a;
  --primary-dark: #0d1a3d;
  --off-white: #1a202c;
  /* ... جميع المتغيرات تعاد تعريفها */
}
```
